# GradPath
🔗 Try GradPath: (https://chatgpt.com/g/g-69bd4728a54c8191b2facacf4a82107a-gradpath-map4jobs)

GradPath is a career guidance assistant designed to help final and pre-final year students prepare for internships and placements.

## Features

* Resume improvement and structuring
* Project evaluation and presentation guidance
* Interview preparation support
* Company-specific preparation insights

## Approach

GradPath focuses on providing clear, actionable guidance tailored to a student’s background, helping them improve their chances of getting shortlisted.

## Tech

* Built using ChatGPT (Custom GPT)
* Structured prompts and curated knowledge base
